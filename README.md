# homebrew-tap

Homebrew tap for schroneko's projects.

## Usage

```bash
brew tap schroneko/tap
```

## Available Formulae

### castkit

Universal model quantization and format conversion CLI supporting GGUF, MLX, GPTQ, and AWQ.

```bash
brew install schroneko/tap/castkit
```

https://github.com/schroneko/castkit

## Available Casks

### NiceVoice

macOS native voice memo app.

```bash
brew install --cask schroneko/tap/nicevoice
```
